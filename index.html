IMPORTANT CONCEPTS

How the internet works:

Browsers send HTTP Requests to a server, which then finds the HTML document and sends the code back to your computer. Your browser then interprets that code and displays it in a user friendly way!

Check out this image I found online for more

HTML Vocabulary

TEXT CONTENT is what you see, whereas
MARKUP is what the text looks like

TAGS tell a browser about a certain ELEMENT

ELEMENTS make up an HTML document

ATTRIBUTE is a property of an HTML element

VOID vs CONTENT

A VOID TAG does not have content,
and so doesn't require a closing tag

This leads up to the distinction between
INLINE or BLOCK elements

INLINE vs BLOCK

As the name suggests INLINE elements are placed in line with the text. HTML will by default place text on a single line
unless you insert a
BREAK

Block elements form a kind of 'box' around themselves, and the content inside of them
